org-mode导出选项的学习

2021-05-07 约 311 字 预计阅读 1 分钟 次阅读

文章目录

导出的设置项

  1. 设置高亮语法转移方式

       #+hugo_code_fence: nil

    1. nil : 转为hugo高亮语法{{}},出现代码块无法左对齐问题

                 #+hugo_code_fence: nil

    2. t:转为markdown代码块高亮```code```

      ```el
#+STARTUP: overview
 ```

  2. Org首次访问文件时使用的启动选项

       #+STARTUP: overview

    overview :只显示一级标题
    content :所有标题
    showal:禁止折叠任何入口
    showeverything:显示抽屉中的内容
    indent: 以打开组织缩进模式开,动态虚拟缩进由变量(setq org-startup-indented t)
    noindent:在组织缩进模式关闭的情况下启动

  3. 禁用下划线转义

    #+OPTIONS: ^:nil

    等价：(setq-default org-use-sub-superscripts nil)

  4. 生成目录

    #+OPTIONS: toc:t
#+OPTIONS: toc:2 num:2

    toc等价(setq org-export-with-toc t)
    num等价(setq org-export-with-section-numbers 2)

    1 header foobar
1.1 header hello
1.2 header welcome
2 header hehe
some text here
2.1 header haha
*low level*

  5. 导出项目计划信息比如schedule time, deadline time等

        #+OPTIONS: p:true
