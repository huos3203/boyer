尝试在shell文件中执行alias别名命令

2019-08-09 约 129 字 预计阅读 1 分钟 次阅读

文章目录

前提: 之前在.zshrc中定义了图片压缩工具命令的别名:

alias pngTo="$hsgToolDir/pngquant/compresspng.py "

在终端直接运行即可: pngTo imgDir

现在打算在hugo 部署脚本中`depley.sh`添加压缩图片的命令: `pngTo `pwd`/static/ox-hugo` 执行过程提示:找不到pngTo命令.

暂时没使用别名,采用了绝对路径执行:

# 压缩图片
echo $hsgToolDir
python $hsgToolDir/pngquant/compresspng.py `pwd`/static/ox-hugo
python $hsgToolDir/pngquant/compresspng.py `pwd`/iPics/images
