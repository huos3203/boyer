DONE 学习spacemacs安装配置和使用

学习基本的配置和安装方式

迁移子龙山人的配置，融会贯通bC-c s：设置开始时间

C-c d：设置结束时间 日格式4 时间格式：5:00

j:跳转日期 r:刷新agenda d：切换到当天日程 w：切换到一周日程

DONE 熟练掌握org-mode管理任务的流程

切换任务状态：`C-c C-t`,TODO/DONE/支持自定义状态

在gtd.org中新建任务和安排日程

  1. `C-c C-s`设置开始时间 `C-c C-d`设置结束时间

  2. DONE 优先级设置/tag管理

org-capture记事本的快捷键`C-c r`的定义和用法

  1. `C-c r`进入任务灵感收集页

  2. 在init-org.el中创建capture模版，会在C-c r启动页面新增选项

org-agenda查看任务面板的快捷键和用法

  1. `C-c a`查看日程安排 d：当天任务 w：切换到一周日程 r：刷新日常 j：跳转日期

  2. 在agenda页面选中任务，启用`C-x org-pomodono`番茄钟
